TripleO

  • using Openstack tools to manage physical machines
  • using that machines to run Openstack itself

Openstack on Openstack

  • Undercloud
  • Overcloud

Undercloud

  • Openstack on baremetal machines
  • self hosted

Overcloud

  • Openstack hosted by Undercloud
  • Heat for orchestration

Tools

  • Nova Baremetal Driver
  • Heat
  • Diskimage-builder
  • OS-config-applier
  • OS-config-refresh

Nova Baremetal Driver

  • like other nova drivers (kvm, xen)
  • but no hypervisor running - tenant gets direct access to physical machine
  • single instance

Heat

  • used for orchestration Overcloud
  • configuration metadata to machines
  • exported metadata from machines
  • triggers

Diskimage-builder

  • github.com/stackforge/diskimage-builder
  • used to build images of machines
  • elements

OS-config-applier

  • github.com/stackforge/os-config-applier
  • cloud-metadata to service config files

OS-config-refresh

  • github.com/stackforge/os-refresh-config
  • simple tool to ensure that configuration is applied on a machine
